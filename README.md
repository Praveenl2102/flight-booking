✈️ Flight Booking Analysis Dashboard

📌 Project Overview

This project presents a Power BI dashboard designed to analyze flight booking data. It provides insights into passenger behavior, booking trends, airline performance, and revenue generation to support better decision-making in the aviation industry.

🎯 Objectives
Analyze flight booking trends over time
Identify peak travel periods and seasonal demand
Understand passenger preferences (class, routes, airlines)
Evaluate cancellation patterns
Track revenue and ticket pricing trends
📊 Key Features
📅 Time-based Booking Analysis (daily, monthly, yearly trends)
🛫 Route Analysis (source → destination insights)
🧑‍🤝‍🧑 Passenger Segmentation
❌ Cancellation Insights
💺 Class-wise Distribution (Economy, Business, First Class)
💰 Revenue & Ticket Price Analysis
📈 Interactive dashboards with slicers and filters
🛠 Tools & Technologies
Power BI – Data visualization
Power Query – Data cleaning & transformation
DAX (Data Analysis Expressions) – Measures and KPIs

📂 Data Processing Steps
Imported raw flight booking data into Power BI
Cleaned and transformed data using Power Query
Created relationships between tables
Developed DAX measures (Revenue, Booking Count, Cancellation Rate)
Designed interactive dashboards

Open using Power BI Desktop
Explore visuals using filters and slicers
📌 Key Insights
Peak travel seasons and high-demand routes
Airline performance comparison
Customer booking behavior
Revenue trends across routes and classes
📊 Sample KPIs
Total Bookings
Total Revenue
Average Ticket Price
Cancellation Rate
Load Factor (Seat Utilization)
